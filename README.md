** Financial Flow 
To use this website user can manege there 
1.Users can add their income or expense details — including type, category, amount, description, and date.
2.Users can see all their transactions in a clean list, sorted by date or amount.
3.If they entered something wrong, users can update existing transaction details — like fixing amount or changing category.
4.Users can remove any transaction they no longer need.
5.Each transaction can be opened in a detailed view showing all info — type, category, amount, description, and date.
